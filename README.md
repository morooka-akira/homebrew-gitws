# Morooka-akira Gitws

## How do I install these formulae?

`brew install morooka-akira/gitws/<formula>`

Or `brew tap morooka-akira/gitws` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "morooka-akira/gitws"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
